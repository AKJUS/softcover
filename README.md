# Polytexnic CLI

Command line interface for Polytexnic.com

## Installation

		$ gem install polytexnic

## Usage

		$ polytexnic <command>

## Commands:

* login
* logout
* build:html, build:pdf, build:epub, build:mobi, build:all
* build (aliased to build:html)
* publish
* new 

## Development Notes
		$ polytexnic config:add host=http://localhost:3000
